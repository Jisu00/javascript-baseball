## 야구 게임

### 기능 목록

1. 게임 시작 문구 출력
2. 사용자 입력
3. 입력 값 유효한지 판별
   > - 3자리 수가 맞는지
   > - number type인지 아닌지
   > - 1부터 9까지의 수가 맞는지
   > - 각 자리 수가 서로 중복되지 않는지
4. 입력 값과 정답 비교
   4-1. 정답이 아닐 경우

   > - 볼 개수 표시
   > - 스트라이크 개수 표시
   > - 하나도 없는 경우 표시

4-2. 정답인 경우

5-1. 2로 돌아감
5-2. 게임 재시작 여부 질문 및 입력

> - 새로 시작
> - 게임 종료
